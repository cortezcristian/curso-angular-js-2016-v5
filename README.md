## Setup Karma

```
$ npm install -g karma-cli # Commandline Interface
$ npm install karma karma-jasmine karma-phantomjs-launcher jasmine-core --save-dev
$ mkdir tests
$ karma init tests/karma.config.js
$ karma start tests/karma.config.js
```
